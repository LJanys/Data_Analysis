Chapter 13 Problem Set 4: NP-Density estimation

Generate a sample of 100 points from a distribution of your choice and program (by hand) the kernel density estimator \[\hat{f}(x)=\frac{1}{nh}\sum_{i=1}^nK\left(\frac{x-X_i}{h} \right)\] using the Epanechnikov Kernel below and plot the resulting function estimate.

\[K(x)=\frac{3}{4}\left(1-|x|^2 \right)\mathbf{1}(|x|\leq 1)\]
